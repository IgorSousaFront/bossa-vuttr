# VUTTR | Bossabox

## Instalação

- Primeiramente faça o fork no repositorio;
- Clone o projeto para a sua máquina;
  ```git clone https://github.com/{username}/bossa-vuttr.git```
- Instale as dependências de desenvolvimento;
  ```$ npm install```
- Inicie o modo de desenvolvimento
  ```$ npm run dev```
